English Adjective Quiz

One-time setup — only you as teacher needs to do this!

This app uses Supabase as a free database so results are saved online and accessible from any device. Follow the steps below — takes about 10 minutes.

Step 1 — Create a free Supabase account

Go to supabase.com and click Start your project. Sign up with your Microsoft account (or email).

Step 2 — Create a new project

Click New project. Give it a name (e.g. adjective-quiz). Choose a region close to Switzerland (e.g. Frankfurt). Set a database password — save it somewhere safe. Click Create new project and wait ~1 minute.

Step 3 — Create the results table

In your Supabase project, click SQL Editor in the left menu. Paste this SQL and click Run:

CREATE TABLE quiz_results (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  student_name text NOT NULL,
  score_pct integer NOT NULL,
  questions_answered integer NOT NULL,
  rule_stats jsonb NOT NULL,
  completed_at timestamp with time zone DEFAULT now()
);

ALTER TABLE quiz_results ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow insert for all"
  ON quiz_results FOR INSERT
  WITH CHECK (true);

CREATE POLICY "Allow select for all"
  ON quiz_results FOR SELECT
  USING (true);
